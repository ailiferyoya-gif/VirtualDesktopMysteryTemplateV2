# Virtual Desktop Mystery Template V2 — Web Preview

仮想デスクトップ型Web謎解きの制作テンプレート V2.1 を、neutral fixturesで操作確認できる公開プレビューです。

- Windows風デスクトップとウィンドウ操作
- Search / Browser / Social / Link / Postbox
- Photos / Audio / Files / Notes / Settings
- ローカル検索履歴、タブ別ブラウザ履歴、添付、通話状態、ごみ箱
- 8種類の架空サイト向けリッチレイアウト
- 360–375px対応

このリポジトリには公開監査済みのWebHosted成果物だけを置いています。制作資料、答え、Evidence設計、監査設定、元fixtureは含みません。表示される人物名・自治体名・投稿・連絡先は、動作確認用の架空データです。

## Local preview

`index.html`を静的サーバーで開いてください。外部API、実通話、実メッセージ送信、認証、位置情報、カメラ、マイクは使用しません。
